# :dragon: gargula :dragon:

A JavaScript package for printing the infamous `gargoyle` or `gargula` in portuguese

This package can also be used in browser
## Installation

`npm install --save gargula`

## Usage

```javascript
const gargula = require('gargula')

gargula()
```
